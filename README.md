# MTG-SYSTEM
Projeto ERP Industrial: Automatizando o Coração da Fábrica

Olá! Me chamo **Guilherme de Jesus Mota** e este é o meu projeto de um sistema **ERP (Enterprise Resource Planning)**. 

Em termos simples: estou construindo uma ferramenta inteligente para que indústrias parem de usar planilhas de papel ou arquivos confusos e passem a controlar todo o seu estoque e parceiros em um só lugar de forma automática.

---

## 🤔 O que este sistema faz?

Imagine uma fábrica que produz peças. Ela precisa saber:
1. **Quanto tem de material no estoque?** (Produtos)
2. **Para quem ela está vendendo?** (Clientes)
3. **De quem ela compra a matéria-prima?** (Fornecedores)

Este sistema organiza esse "triângulo" de informações para que nada falte na linha de produção.

---

## 🏗️ Como ele é construído? (Explicando as peças)

Para quem não é da área, um sistema é como um carro:

* **O Visual (A Carroceria):** Desenvolvido com **HTML, CSS e JavaScript**. É a parte que o usuário vê, clica e interage. É aqui que os botões são bonitos e as tabelas são fáceis de ler.
* **O Motor (O Backend):** Feito em **C# (C-Sharp)**. É a parte "invisível" que faz os cálculos, valida se o CNPJ está certo e decide o que deve ser salvo.
* **O Cofre (Banco de Dados):** Utilizamos tecnologia **NoSQL**. Imagine um arquivo super moderno e rápido onde guardamos todas as informações da empresa com máxima segurança.

---

## 📈 O Passo a Passo do Projeto

Minha jornada de desenvolvimento segue estas etapas:

1.  **Planejamento:** Entender o que uma indústria precisa no dia a dia.
2.  **Design e Front-end:** Criar as telas onde o funcionário vai cadastrar os produtos.
3.  **Construção do Motor:** Programar em C# para que os dados saiam da tela e cheguem ao cofre.
4.  **Conexão Total:** Fazer com que o visual e o banco de dados "conversem" sem erros.
